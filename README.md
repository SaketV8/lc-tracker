## 🛠️ The Tech Stack

- **Logic:** Vanilla JavaScript (ES6+)
- **Build Tool:** [Vite](https://vite.dev/) (Used for lightning-fast bundling and production minification)
- **Styling:** Custom CSS3 (Modular & Responsive)
- **Data Source:** Company-wise mappings sourced from [kash04x/BetterLC](https://github.com/kash04x/BetterLC)
- **Hosting & Analytics:** [Cloudflare](https://cloudflare.com/)

## ✨ Key Features

- **Company-Specific Curation:** Easily find which problems are being asked at top tech firms.
- **Performance First:** No heavy frameworks (React/Vue). This project is built with "raw" web technologies for near-instant load times.
- **Clean UI:** A minimalist interface designed for focus and productivity.
